# dotfiles
My dotfile configuration

## Shell aliases

### Claude Code
| Alias | Command | Description |
|-------|---------|-------------|
| `cc` | `claude` | Start Claude Code |
| `ccc` | `claude --continue` | Continue last session |
| `ccr` | `claude --resume` | Resume a previous session |
