# Keyboard Shortcuts Reference

## System

| Shortcut | Action |
|---|---|
| `OPT + Space` | Raycast |
| `CMD + Space` | Spotlight |
| `CMD + SHIFT + V` | Clipboard history (Raycast) |
| `CTRL + 1/2/3` | Switch desktop |
| `CMD + TAB` | Switch windows (Alt-Tab) |
| `CMD + \`` | Switch windows within same app |
| `CMD + ,` | Preferences in any app |
| `CTRL + UP` or `F3` | Mission Control |
| `CMD + SHIFT + 4` | Screenshot region |
| `CMD + SHIFT + 5` | Screenshot menu |

## Finder

| Shortcut | Action |
|---|---|
| `CMD + SHIFT + H` | Home folder |
| `CMD + SHIFT + G` | Go to path |
| `CMD + SHIFT + .` | Show hidden files |
| `SPACE` | Quick Look |
| `CMD + DELETE` | Move to trash |

## Terminal

| Shortcut | Action |
|---|---|
| `CTRL + R` | Fuzzy history search (fzf) |
| `CTRL + T` | Fuzzy file find (fzf) |
| `CTRL + A` | Start of line |
| `CTRL + E` | End of line |
| `CTRL + U` | Clear line |
| `CTRL + L` | Clear screen |
| `OPT + ŌåÉ/ŌåÆ` | Skip words |

## Zed

### App commands

| Shortcut | Action |
|---|---|
| `CMD + P` | Quick open file |
| `CMD + SHIFT + P` | Command palette |

| `CMD + R` | Toggle right dock (Agent Panel by default) |

### Vim motions (default: `vim_mode: true`)

| Key | Action |
|---|---|
| `h / j / k / l` | Left / down / up / right |
| `w / b` | Next / previous word |
| `gg / G` | Top / bottom of file |
| `i / a / o` | Insert / append / new line below |
| `ESC` | Exit insert mode |
| `/` | Search forward |
| `:w` | Save buffer |
| `:q` | Close buffer |
| `dd / yy / p` | Delete / yank / paste line |
