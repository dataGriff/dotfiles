# Keyboard Shortcuts Reference

## System

| Shortcut | Action |
|---|---|
| `OPT + Space` | Raycast |
| `CMD + Space` | Spotlight |
| `CMD + SHIFT + V` | Clipboard history (Raycast) |
| `CTRL + 1/2/3` | Switch desktop |
| `CMD + TAB` | Switch windows (Alt-Tab) |
| `CMD + \`` | Switch windows within same app |
| `CMD + ,` | Preferences in any app |
| `CTRL + UP` or `F3` | Mission Control |
| `CMD + SHIFT + 4` | Screenshot region |
| `CMD + SHIFT + 5` | Screenshot menu |

## Finder

| Shortcut | Action |
|---|---|
| `CMD + SHIFT + H` | Home folder |
| `CMD + SHIFT + G` | Go to path |
| `CMD + SHIFT + .` | Show hidden files |
| `SPACE` | Quick Look |
| `CMD + DELETE` | Move to trash |

## Terminal

| Shortcut | Action |
|---|---|
| `CTRL + R` | Fuzzy history search (fzf) |
| `CTRL + T` | Fuzzy file find (fzf) |
| `CTRL + A` | Start of line |
| `CTRL + E` | End of line |
| `CTRL + U` | Clear line |
| `CTRL + L` | Clear screen |
| `OPT + ←/→` | Skip words |

## VS Code

| Shortcut | Action |
|---|---|
| `CMD + P` | Quick open file |
| `CMD + SHIFT + P` | Command palette |
| `CMD + B` | Toggle sidebar |
| `CMD + J` | Toggle terminal panel |
| `CMD + K, Z` | Enter Zen mode (`CMD + K` is a chord — press and release it, then press `Z`) |
| `ESC, ESC` | Exit Zen mode |
| `CMD + D` | Select next occurrence |
| `OPT + ↑/↓` | Move line |
| `CMD + SHIFT + K` | Delete line |
| `CMD + /` | Toggle comment |
| `F12` | Go to definition |
| `OPT + F12` | Peek definition |

## Chrome + Vimium

| Key | Action |
|---|---|
| `F` | Click any element |
| `H / L` | Back / forward |
| `J / K` | Scroll down / up |
| `G G` | Jump to top of page |
| `SHIFT + G` | Jump to bottom |
| `SHIFT + J / K` | Switch tabs |
| `/` | Search page |
| `T` | Open new tab |
| `X` | Close current tab |

## Linear

| Shortcut | Action |
|---|---|
| `C` | Create new issue |
| `CMD + K` | Command palette |
| `1 / 2 / 3` | Set priority |
| `A` | Assign issue |
| `S` | Change status |
| `F` | Filter |
